# 🚀 Java DSA

A comprehensive repository for learning and mastering Data Structures and Algorithms using Java. This project contains implementations of fundamental and advanced DSA concepts, coding challenges, and interview preparation materials.

## 📚 Topics Covered

### Arrays

* Traversal
* Searching
* Sorting
* Prefix Sum
* Sliding Window
* Two Pointers

### Strings

* String Manipulation
* Pattern Matching
* Palindrome Problems
* Anagram Problems

### Linked Lists

* Singly Linked List
* Doubly Linked List
* Circular Linked List

### Stacks & Queues

* Stack Implementation
* Queue Implementation
* Monotonic Stack
* Priority Queue

### Hashing

* HashMap
* HashSet
* Frequency Counting

### Trees

* Binary Tree
* Binary Search Tree (BST)
* Tree Traversals
* Lowest Common Ancestor

### Heaps

* Min Heap
* Max Heap
* Heap Sort
* Priority Queue Applications

### Graphs

* BFS
* DFS
* Topological Sort
* Shortest Path Algorithms
* Union Find (DSU)

### Recursion & Backtracking

* Recursive Problems
* Subsets
* Permutations
* N-Queens

### Dynamic Programming

* Memoization
* Tabulation
* Knapsack
* Longest Common Subsequence
* DP on Trees

### Greedy Algorithms

* Activity Selection
* Interval Scheduling
* Huffman Coding

### Advanced Topics

* Trie
* Segment Tree
* Fenwick Tree
* Disjoint Set Union
* Bit Manipulation

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/IncridableAcuman/java-dsa.git
cd java-dsa
```

### Run Examples

Compile:

```bash
javac Main.java
```

Run:

```bash
java Main
```
